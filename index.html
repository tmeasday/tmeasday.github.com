Work / Side Projects

I'm a web developer with my finger in a few pies.

  • I run Percolate Studio, a web application development studio.
  • Bindle is a site to make beautiful visual lists of what's important to you.
  • CrowdSpot is a map based survey tool to connect decision makers with the community.

Meteor

I'm pretty heavily involved in the Meteor community.

  • I helped create Meteorite and Atmosphere, tools that let developers share and use each other's meteor smart packages.
  • I maintain an unofficial FAQ, which give opinionated answers to a bunch of common questions.
  • I worked with Sacha Grief to create Telescope, a Hacker News clone "framework", about which we are writing an eBook.
  • We (Percolate Studio) created League, a sports management app in meteor.
